Dodo Pizza API (Work in Progress 🚧)
Это учебный проект для отработки навыков построения RESTful сервисов на Spring Boot 3.

Что уже реализовано:
Спроектирована многослойная архитектура (Controller -> Service -> Repository).
Настроена глобальная обработка исключений.
Внедрен маппинг сущностей через MapStruct.
Подключена автодокументация Swagger UI.

В планах (Roadmap):
Подключение PostgreSQL и миграций Liquibase.
Написание Unit и Integration тестов.
Внедрение Spring Security.
