# Dodo Pizza Project (Backend) 🍕

Учебный проект REST API для сервиса доставки еды. Реализован с соблюдением стандартов промышленной разработки.

## Технологический стек
* **Java 17** & **Spring Boot 3**
* **Gradle** (система сборки)
* **PostgreSQL** (база данных)
* **Liquibase** (миграции БД)
* **MapStruct** (маппинг DTO)
* **Docker & Docker Compose** (контейнеризация)
* **Swagger UI** (документация API)

## Основные фичи
- Полный CRUD для управления товарами и категориями.
- Пагинация и динамическая сортировка товаров.
- Глобальная обработка исключений и валидация данных.
- Логирование операций и ошибок (SLF4J + Logback).
